# System Requirements

Minimum Per Node:

CPU: 2 vCPU (4 recommended for masters)
RAM: 4 GB (8 GB for masters recommended)
Disk: 40 GB
OS: RHEL / Rocky / Alma / CentOS 8/9
Network: Static IP required

Ports Required:

6443 - Kubernetes API
2379-2380 - etcd
10250 - kubelet
30000-32767 - NodePort range
